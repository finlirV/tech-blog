# 👋 关于 Tech Insights

欢迎来到 Tech Insights，一个专注于技术深度分析与工程实践的个人博客。

## 🎯 博客定位

Tech Insights 致力于：

- **深度技术分析**：不仅仅是 How，更是 Why 和 How to think
- **工程实践分享**：从架构设计到代码实现的全链路思考
- **前沿技术探索**：跟踪最新技术趋势并给出实践建议
- **学习路径规划**：为不同阶段的技术人提供成长指南

## 📚 内容方向

### 1. 前端技术栈
- 现代前端框架（React、Vue、Svelte）深度解析
- 工程化与构建工具最佳实践
- 性能优化与用户体验
- TypeScript 高级应用

### 2. 架构设计
- 微前端架构实践
- 服务端渲染与同构应用
- 状态管理方案对比
- 组件库设计与维护

### 3. 工程实践
- CI/CD 流水线设计
- 测试策略与质量保障
- 代码规范与团队协作
- 技术选型方法论

### 4. 技术思考
- 技术发展趋势分析
- 学习与成长方法论
- 技术团队管理经验
- 开源项目参与指南

## 🛠️ 技术栈

本博客采用以下技术构建：

- **静态站点生成器**：[Hexo](https://hexo.io/)
- **主题**：[Butterfly](https://butterfly.js.org/)
- **部署平台**：[Netlify](https://www.netlify.com/)
- **评论系统**：[Giscus](https://giscus.app/)（基于 GitHub Discussions）
- **全文搜索**：[Algolia](https://www.algolia.com/)
- **代码高亮**：[Prism.js](https://prismjs.com/)
- **图表渲染**：[Mermaid](https://mermaid.js.org/)

## 🌈 视觉设计

博客设计灵感来源于 [overreacted.io](https://overreacted.io/)，融合了：

- **平滑动画过渡**：悬停放大、链接下划线、按钮涟漪
- **深色/亮色双主题**：自动跟随系统偏好
- **响应式布局**：完美适配移动端与桌面端
